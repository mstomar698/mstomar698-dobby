## Doby-Ads-Assignment

### How to run the code

```sh

# Clone the repository
git clone https://github.com/mstomar698/mstomar698-dobby.git
# Start Server
cd backend
npm install
npm start
# Start Client
cd ../frontend
npm install
npm start
```

### Run locally

```sh
# In main directory
npm run install
npm run build
# Update .env
npm run serve
# change terminal
npm run start
```
